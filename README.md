# foodie
系统包括三个部分，硬件，服务器后端，微信小程序客户端。<br>
## 硬件部分
主要采集用户用餐数据，并通过TCP协议上传到服务器端。<br>
## 服务器后端
处理硬件发来的数据，提供了几个接口应对小程序的请求。<br>
## 微信小程序
提供与用户交互的功能：饮食分析、食堂菜品情况查询、饮食周报、饮食定制（待开发）、反馈、社交圈<br>
